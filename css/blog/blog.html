A tipagem de dados do javascript e o que é typescript

Typescript ou muitas vezes chamado de ts é um super set extremamente importante para os frameworks ou librarys javascript pois ele dá uma coisa extremamente importante para linguagens atuais, A tipagem de dados ou tipagem dinâmica com ela podemos contornar um dos piores problemas da linguagem javascript e de brinde adicionando a famosa poo ou oop sigla para Programação Orientada a Objeto
